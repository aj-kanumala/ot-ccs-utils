# ot-ccs-utils
A lightweight toolkit for solving optimal transport problems. 

### Setup

#### Step 1 - Create the virtual environment (only needed once):

```python
python3 -m venv venv 
```

#### Step 2 - Activate it:

```python
source venv/bin/activate
```
Once activated, your terminal prompt will change to show `(venv)` at the start:


#### Step 3 - Install required packages in virtual environment:

```python
pip install git+https://github.com/aj-kanumala/ot-ccs-utils.git
```

#### Other useful commands:

```python
# Deactivate venv when you're done
deactivate

# Install packages while inside venv
pip install numpy pandas matplotlib

# Install all from requirements.txt
pip install -r requirements.txt
```
